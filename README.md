### 3. 项目编译说明：

   1 .`3rd\CxImage\cximage701_full\CxImgLibVC2019.sln` 首先编译这个目录。注：新版本已经去除CxImage支持，不使用的话，可以不编译。 

2. 然后在编译当前目录  优先选择UnicodeDebug 编译项目

    
   
   ### 4. 重要说明

设计器创建控件和修改属性依据`bin\DuiLib.xml`，理论上只要遵循这个约定的duilib版本都可以使用这款设计器。
